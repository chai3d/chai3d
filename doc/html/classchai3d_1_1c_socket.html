◆ getLocalIP()

std::string chai3d::cSocket::getLocalIP ( )
static

This method retrieves the local IP address. If the system is connected to the network, this method returns the first non-loopback IP address available on the system. Otherwise, the loopback adapter address (usually 127.0.0.1) is returned.

Returns
a string containing the local IP on success, the loopback adapter address otherwise.

◆ peek() [1/2]

int chai3d::cSocket::peek ( char *  a_output,
unsigned int  a_size 
)

This method reads any available data on a socket into a given buffer, but without emptying the buffer. The size of the buffer returned depends on the buffer size and the amount of available data (whichever is smaller). If the BLOCKING attribute is set, the call blocks until data is available. Otherwise, the call returns immediately if no data is available.

Parameters
a_outputA buffer to collect the data requested.
a_sizeThe size of the a_output buffer in bytes.
Returns
The number of bytes read on success, TIMEOUT if the socket read operation timed out, WOULDBLOCK if the socket is non-blocking and there is no data available yet, -1 on error.
