• Oculus module documentation

The Oculus Rift is a virtual reality system that completely immerses the user inside virtual worlds. CHAI3D supports for the Oculus Rift through a module based on API version 0.8 (beta) of the display. The current implementation is limited to the Windows operating system, but support for Linux and Mac OS X is expected once the product rolls into production. Since the Oculus Rift is still under development, please check their website for more updated information about the product.
